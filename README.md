# IMS Projekt-Anfrage

Lead-Generierung für Architekten & Planer. One-Page Progressive Disclosure Flow.

**Stack:** Next.js 16 + Tailwind + Supabase + Vercel
**Domain:** projekt.ihr-moebel-schreiner.de

## Setup
1. Run `supabase-setup.sql` in Supabase SQL Editor
2. Copy `.env.example` to `.env.local` and fill in keys
3. `npm install && npm run dev`
